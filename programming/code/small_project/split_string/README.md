# Split String

## Usage

Command Format:

```bash
> python3 split.py
```

Example:

```bash
> python3 split.py
Please enter string: .p-0{padding:0!important}.p-1{padding:.25rem!important}.p-2{padding:.5rem!important}
Please enter number of delimiter(s): 2
Please enter delimiter(s): { }
Please enter table header(s): Class Content

+-------+--------------------------+
| Class |         Content          |
+-------+--------------------------+
|  .p-0 |   padding:0!important    |
|  .p-1 | padding:.25rem!important |
|  .p-2 | padding:.5rem!important  |
+-------+--------------------------+
```
