# Speaker

Using TTS technology to read Chinese and English.

## Usage

Command Format:

```bash
> python3 main.py data_file_name
```

Example:

1.  Your data file format should be as follows:

    ```txt
    | English           | Chinese        |
    | ----------------- | -------------- |
    | Hello             | 你好           |
    | How are you?      | 你好吗？       |
    | What's your name? | 你叫什么名字？ |
    ```

2.  Execute `python3 main.py data.txt`.
