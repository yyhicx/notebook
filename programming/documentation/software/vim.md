# Vim

## Linux

Running configuration: copy src/.vimrc to ~/.vimrc

## Windows

Running configuration:

*   open gvim
*   编辑 -> 启动设定（或者:vi $vim/_vimrc）
*   copy src/.vimrc to $vim/_vimrc

禁止VIM生成备份文件：

*   open gvim
*   编辑 -> 启动设定（或者:vi $vim/_vimrc）
*   set nobackup
*   set noundofile
*   set noswapfile
