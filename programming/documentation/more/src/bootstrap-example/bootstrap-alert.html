提示框

通过 .alert 类，后面加上特定意义的颜色类来实现：

alert-primary：重要的文本。
alert-secondary：次要的文本。
alert-light：浅灰色文本（白色背景上看不清楚）。
alert-dark：深灰色文本。
alert-success：执行成功的文本。
alert-info：代表一些提示信息的文本。
alert-warning：警告文本。
alert-danger：危险操作文本。
